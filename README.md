# A股复盘系统

一个专业的A股盘后复盘工具，支持多种交易模式选股、行情查看、交易记录分析和复盘笔记。

## 功能模块

### 📊 大盘情绪
- 四大指数实时行情（上证、深证、创业板、科创50）
- 涨跌家数统计与可视化
- 涨跌停比分析
- 资金流向监测
- 板块热度热力图
- 市场情绪评分

### 🔍 模式选股（5大交易模式）

#### 1. 趋势为王
- 均线多头排列持续天数筛选
- 最低价 > 10日均线天数筛选
- 排除银行股
- 5日换手率 < 100%
- 综合评分排序

#### 2. 断板反包
- 首板涨停后5天观察期
- 第5天必须在开盘价-收盘价区间内
- 加分因子：
  - 第二天早盘10点前冲高
  - 第二、三天有下影线
  - 首板强度（一字板/秒板/普通涨停）
- **自选拓展池**：手动添加第4/5天在范围内但过期的股票

#### 3. 龙回头
- 手动维护龙头股票池
- 记录第一波涨幅和回落幅度
- 跟踪回落状态，提示介入时机

#### 4. 主升踏浪
- 5/10/20日强度排名
- 综合评分自动计算
- 一键添加自选

#### 5. 博主跟踪
- 管理关注的博主列表
- 记录博主推荐的股票
- 跟踪推荐后涨幅和启动状态

### 📈 行情中心
- K线图（支持MA20/MA60均线）
- 分时图（带均价线）
- 成交量显示
- 股票搜索

### 💼 交易记录
- CSV/Excel文件导入
- 自动计算盈亏和胜率
- 交易统计面板
- 完整的交易历史记录

### 📝 复盘笔记
- 创建每日复盘笔记
- 本地存储，数据持久化
- 支持标题、日期、内容
- 笔记列表管理

## 技术特点

- **纯前端实现**：HTML/CSS/JavaScript，无需后端服务器
- **深色专业主题**：护眼设计，金融终端风格
- **响应式布局**：支持电脑和手机访问
- **本地数据存储**：使用localStorage保存所有数据
- **模拟数据演示**：内置模拟数据，开箱即用

## 使用说明

### 本地使用
1. 直接用浏览器打开 `index.html` 文件
2. 所有数据保存在浏览器本地存储中

### 数据导入
- 交易记录支持CSV格式导入
- 可从券商导出交易记录后导入分析

### 数据持久化
- 所有数据自动保存到浏览器localStorage
- 清除浏览器数据会导致数据丢失
- 建议定期导出重要数据备份
