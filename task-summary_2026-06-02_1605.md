# A股复盘系统 - 开发完成

## 目标
为江铂打造一个专业的A股盘后复盘网站，支持5种交易模式选股、行情查看、交易记录分析和复盘笔记。

## 关键决策
1. **技术方案**：纯静态网站（HTML/CSS/JS），无需后端，免费部署到GitHub Pages
2. **数据来源**：先用模拟数据演示功能，后续可接入Tushare/AkShare免费API
3. **视觉风格**：深色专业主题，金融终端风格，护眼设计
4. **数据持久化**：使用localStorage本地存储，所有数据保存在浏览器中

## 实现的功能模块

### 📊 大盘情绪
- 四大指数行情卡片（上证、深证、创业板、科创50）
- 涨跌家数可视化进度条
- 涨跌停比统计
- 资金流向柱状图
- 板块热度热力图
- 市场情绪评分仪表盘

### 🔍 模式选股（5大交易模式）
1. **趋势为王**：均线多头排列、换手率筛选、综合评分
2. **断板反包**：首板后5天观察、加分因子计算、自选拓展池
3. **龙回头**：手动维护龙头池、跟踪回落状态
4. **主升踏浪**：多周期强度排名
5. **博主跟踪**：博主管理、推荐股票跟踪

### 📈 行情中心
- K线图（支持MA20/MA60均线、成交量）
- 分时图（带均价线）
- 股票搜索功能

### 💼 交易记录
- CSV文件导入
- 盈亏统计面板（总盈亏、胜率、交易次数）
- 完整交易记录表格

### 📝 复盘笔记
- 创建/编辑/删除笔记
- 笔记列表管理
- 本地持久化存储
